# 📧 Email Automation System

## 🚀 Overview

This is a PHP-based email automation system using PHPMailer and SMTP.

Este es un sistema de automatización de correos usando PHP y SMTP.

---

## 🎯 Features | Funcionalidades

- Contact form email sending
- CV submission system
- SMTP Gmail integration
- Secure configuration system
- JSON API responses

---

## 🛠️ Technologies

- PHP
- PHPMailer
- SMTP (Gmail)
- JSON API

---

## 📂 Structure

- api → backend endpoints
- config → secure configuration
- templates → email designs
- phpmailer → library

---

## ⚙️ Setup

1. Clone repository
```bash
git clone https://github.com/your-user/email-automation-system.git
2. Create config file
cp config/config.example.php config/config.php
3. Add SMTP credentials
3. Run on XAMPP / WAMP

Credentials are NOT uploaded
Sensitive files are ignored
